# Mean-Median-Mode-Calculator

Mean Median Mode Calculator

A simple web application that calculates:

Mean
Median
Mode

from a dataset entered by the user.

Built using:

HTML
CSS
JavaScript
Features

✅ Calculate Mean

✅ Calculate Median

✅ Calculate Mode

✅ Step-by-step explanations

✅ Input validation

✅ Responsive UI

✅ Animated result display using setTimeout()


Preview

Users enter numbers separated by commas:

12,21,23,34,43,56,56,67,71

The app then calculates:

Mean
Median
Mode

with detailed working steps.
